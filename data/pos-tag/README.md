# PoS tagging datasets


The datasets consist of sentences and tags as such: 
word_1 word_2 ... word_n \t tag_1 tag_2 ... tag_n


## Sources
- The original files are from [universaldependencies](https://universaldependencies.org/) project. 
Each language will have an Attribution.txt file to the exact Github repo of the original file.
